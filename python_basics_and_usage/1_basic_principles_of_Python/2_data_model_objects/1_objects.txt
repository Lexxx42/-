# все данные находятся в оперативной памяти
# объекты - представление данных в языке python

# Объект  - абстракция для данных
# 1, 2, 3 - числа
# [1, 2, 3] - списки
# "привет" - строки

# Если бы оперативная память была складом, то объекты - это коробки:
# одна коробка с 1, вторая с 2, третья коробка с 3
# коробка с листом
# коробка со строкой

# Сперва мы бы данные поместили в коробку, а только после этого поместили на склад

# Все данные в Python представленны объектами и отношениями между объектами

# Наглядно демонстрируется списком
# a = 1
# b = 2
# с = [a; b]

# В оперативной памяти помимо объектов для 1, 2 и списка, мы должны хранить отношение включения.
# мы знаем, что список с включает в себя а и b

# У объекта в языке Python есть 3 обязательные вещи: идентификатор, тип и значение
